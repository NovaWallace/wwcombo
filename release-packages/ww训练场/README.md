# 鸣潮训练场 使用说明

鸣潮训练场是一个连段练习工具，用来记录自己的操作、整理成连段谱，并在练习时显示置顶连段图提示。它不会替你自动按键，主要作用是提示、记录和判定你的真实输入。

## 快速开始

1. 解压压缩包。
2. 双击运行 `wwcombo.exe`。
3. 第一次使用建议先进入左侧的“设置”，确认按键是否符合自己的游戏键位。
4. 进入“记录”，录一段自己的连招。
5. 保存连段谱后，进入“练习”开始训练。

## 记录连段

进入“记录”页面后：

1. 点击“开始记录 F”，或直接按 `F` 开始。
2. 在游戏里正常打一遍连招。
3. 按 `Esc` 结束录制。
4. 点击“覆盖”，把本次录制载入到连段谱编辑区。
5. 在“连段谱编辑”里检查时间轴，必要时调整指令位置、持续时间、时段和角色轨道。
6. 点击“保存连段谱”，保存到历史连段谱。

如果你已经有连段谱，也可以用“调试”把一轮实战输入合并进去，用来微调容错时间。

## 编辑连段谱

连段谱编辑分为两个模式：

- “时间”：调整每个操作的开始时间、持续时间、所属角色和轨道，也可以添加启动轴、循环轴、自由开火等时段。
- “内容”：修改连段图上实际显示的文字，例如把默认招式名改成更容易看懂的“技能”“解放”“跳”等。

常用操作：

- “添加指令”：在时间轴上手动补一个操作块。
- “添加时段”：添加启动轴、循环轴或自由开火时段。
- 右键操作块：复制、粘贴、删除、切换自由标签等。
- 框选多个操作块：可以一起移动或批量修改部分属性。
- “快捷输入”：按顺序快速填写连段图显示文字。

提示：循环轴的第一个操作建议是切人或共鸣解放，否则时间轴会用红色提示该循环轴可能不合理。

## 练习模式

进入“练习”页面后，选择一种模式：

- “演示”：按时间自动播放整套流程，适合先看连段顺序。
- “练习”：读你的真实输入，输入正确就进入下一步；部分独立动作会按时间播放。
- “挑战”：更接近实战判定，对输入时机要求更高。

按钮说明：

- “开始 F”：开始本轮练习。
- “结束 Esc”：结束当前练习。
- “快捷输入”：快速改连段图显示文字。
- 眼睛图标：打开或关闭置顶连段图。
- “轴首招启动”：开启后，演示/挑战进入启动轴或循环轴时，需要先按该轴内第一个关键操作才会继续播放。

练习时，连段图会显示当前操作、下一步提示、错误反馈和进度。打开“预提示”后，会显示下一步三角箭头和操作文字提示。

## 置顶连段图

置顶连段图用于在游戏或其他窗口上方显示连段提示。

在“练习”页面或“外观”页面点击眼睛图标即可显示/隐藏。

在“外观”页面可以设置：

- 横排或竖排。
- 移动位置。
- 复位位置。
- 胶囊模式或底图模式。
- 角色头像、角色名、颜色。
- 渐影效果。
- 预提示开关。
- 连段图大小、间距、文字样式等。

位置和长宽会分别保存横排、竖排两套设置，下次打开会沿用上次的位置。

如果置顶图关不掉，先确认是否开了旧版本程序。新版关闭主窗口时会自动清理置顶窗口。

## 图标转换

在“内容”模式下可以开启“图标转换”。开启后，练习预览和置顶连段图会把指定文字转换成图片图标。

默认支持的输入示例：

- `跳` 或 `j`：跳跃图标。
- `变奏` 或 `b`：变奏图标。
- `延奏` 或 `y`：延奏图标。
- `S` / `D` / `闪`：长按闪避图标。
- `z` / `Z`：长按普攻，也就是重击图标。
- `E`：长按技能图标。
- `Q`：长按声骸图标。
- `R`：长按共鸣解放图标。
- `J`：长按跳跃图标。
- `i` / `ii` / `iii`：罗马数字图标。

点击“图标转换”旁边的设置按钮，可以自己更换图标图片，也可以修改对应文字。多个文字用 `/` 隔开。

关闭“图标转换”后，连段图会恢复显示普通文字。

## 导入和导出

在“历史连段谱”区域可以：

- 导出当前连段。
- 导出全部连段。
- 导入 JSON。
- 删除历史连段。

导出的 JSON 会保存连段相关的重要信息，包括：

- 指令时间、持续时间、角色、轨道。
- 启动轴、循环轴、自由开火等时段。
- 内容模式里自定义的显示文字。
- 图标转换设置和自定义图标映射。
- 连段图外观设置。
- 置顶连段图横排/竖排的位置和大小。
- 练习里的轴首招启动开关。
- 招式和键位配置。

可以用 JSON 备份自己的连段，也可以发给别人导入测试。

## 键位和权限

- 程序会监听你的真实键盘和鼠标输入，用来判断练习是否正确。
- 如果游戏以管理员权限运行，训练场可能也需要用管理员权限运行，才能稳定收到按键。
- 程序不会自动向游戏发送按键。
- 如果全局监听不可用，可以先确认安全软件、系统权限或游戏运行权限。

## 给测试者的建议

1. 先用自带键位录一小段简单连招，确认记录正常。
2. 再打开置顶连段图，测试横排和竖排显示。
3. 修改一次内容文字，导出 JSON，再重新导入确认文字还在。
4. 测试关闭主窗口后，置顶连段图是否会一起消失。
5. 如果遇到问题，记录发生在哪个页面、做了什么操作、是否打开了置顶连段图。
