# 鸣潮训练场

一个面向《鸣潮》的桌面端连段训练工具。项目基于 Tauri 2、React、Vite 和 TypeScript，主要用于记录连段、编辑连段图、置顶显示操作提示，并通过不同练习模式辅助训练。

## 主要功能

- 连段录制：监听键位输入并生成连段步骤。
- 连段编辑：支持时间轴、循环轴、招式块、内容名称等编辑。
- 自定义内容：可以通过“内容”功能修改招式显示名称，导出 JSON 时会保留这些自定义名称。
- 三种模式：
  - 演示：按时间自动播放，用于查看完整流程。
  - 练习：正确输入后进入下一步，部分独立/自由开火动作按时间播放。
  - 挑战：更接近实战判定，需要按正确时机输入。
- 置顶连段图：支持底图模式、胶囊模式、横排/竖排显示、移动和缩放。
- 预提示：可显示下一步三角提示、文字提示，并让下一步在渐影时保持清晰。
- 设置面板：集中管理外观、提示、键位、招式和连段图相关配置。
- JSON 导入导出：支持导出当前连段或全部连段，保留连段步骤、循环轴、招式引用、自定义内容名称和必要键位/招式信息。

## 本地开发

先安装依赖：

```bash
npm.cmd install
```

启动前端预览：

```bash
npm.cmd run dev
```

启动 Tauri 桌面端：

```bash
npm.cmd run tauri:dev
```

## 打包

生成正式构建和安装包：

```bash
npm.cmd run tauri:build
```

打包产物通常位于：

```text
src-tauri/target/release/
src-tauri/target/release/bundle/
```

其中 `src-tauri/target/release/wwcombo.exe` 是主程序，可用于免安装测试；`bundle` 目录下会包含安装包，例如 NSIS 或 MSI 产物，具体取决于本机 Tauri 打包环境。

## 分发测试说明

- 如果测试者不想安装，可以先尝试直接运行 `wwcombo.exe`。
- 如果使用安装包，部分 Windows 安装器可能会请求管理员权限；这取决于安装器类型和系统策略，不影响免安装 exe 的运行方式。
- 全局按键监听可能受系统权限、安全软件或游戏运行权限影响。如果游戏以管理员权限运行，训练工具也可能需要以相同权限运行才能收到按键。
- 置顶连段图可在设置里切换显示样式、方向、预提示和渐影等选项。
- 导出的 JSON 可以用于备份或迁移连段；通过“内容”自定义过的招式名称会一起保留。
